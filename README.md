# Online Retail II - Exploratory Data Analysis (EDA)

## Project Overview

This project presents an Exploratory Data Analysis (EDA) of the Online Retail II dataset to understand customer purchasing behavior, product performance, sales trends, and revenue distribution. The analysis provides meaningful business insights that can support future decision-making and the development of a recommendation system.

---

## Objectives

* Load and explore the dataset
* Display dataset shape, columns, and data types
* Identify missing values and duplicate records
* Analyze the top 10 best-selling products by quantity
* Analyze the top 10 products by revenue
* Analyze sales performance by country
* Visualize monthly revenue trends
* Create a correlation heatmap
* Detect outliers using box plots
* Generate business insights from the analysis

---

## Dataset

The analysis uses the **Online Retail II** dataset, which contains real transaction records from a UK-based online retailer between 2009 and 2011. The dataset includes invoice numbers, product descriptions, quantities, prices, customer IDs, invoice dates, and countries.

Dataset Source: https://archive.ics.uci.edu/ml/datasets/Online+Retail+II

---

## Analysis Performed

* Dataset Overview
* Missing Value Analysis

* Top 10 Best-Selling Products by Quantity

---

## Key Business Insights

* White Hanging Heart T-Light Holder is the highest-selling product by quantity.
* Regency Cake Stand 3 Tier generates the highest overall revenue.
* The United Kingdom contributes the largest share of total revenue.
* Monthly revenue shows noticeable fluctuations, indicating seasonal purchasing patterns.
* The dataset contains missing values, duplicate records, and outliers that should be addressed before building predictive models.
